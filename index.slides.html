Obiettivi:¶

  • illustrare nuovi strumenti e modalit con cui manipolare i dati

Dati geografici¶

Esistono differenti e numerose codifiche per esprimere i dati spaziali.

Il Sistema di Riferimento Globale (ETRS89) viene indicato, a volte, con lacronimo WGS84.

Per approfondire si consiglia di consultare questo link

Le sigle SRS (spatial reference system) e CRS(coordinate reference system) sono sinonimi.

I dati geografici per poter essere visualizzati dai tool che la piattaforma mette a disposizione devono rispettare la codifica WGS84.

Nel dataset delle strutture ricettive non indicato il tipo di formato usato.

In questo caso necessario capire in quale formato sono espressi e convertire in WSG84

Per effettuare "operazioni geografiche" sui dataset abbiamo a disposizione la libreria geopandas

DAF invio dataset con SFTP¶

Dopo aver metadato il dataset, necessario inviare nella cartella specificata nella scheda informativa il dataset completo.

Puoi utilizzare comodamente alcuni tool come Filezilla oppure farlo attraverso alcune righe di codice.

Dovrai inserire il tuo utente, la tua password e autenticarti via SFTP a daf.teamdigitale.it (porta 22)

Importiamo il modulo pysftp che ci permette di effettuare il trasferimento via SFTP

Consulta la documentazione

In [77]:

In [ ]:

sftp_user = "user"

sftp_path = "test/2"

cnopts = pysftp.CnOpts()
cnopts.hostkeys = None
cnopts.compression = True

with pysftp.Connection(host=sftp_host, 
                       username=sftp_user,
                       password=sftp_password, 
                       cnopts=cnopts) as sftp:
    sftp.chdir(sftp_path)
    sftp.put(local_path, confirm=True)

A questo punto, dopo aver atteso alcuni minuti, il tuo dataset verr caricato e potrai monitorare lo status tramite la scheda del dataset.

Inoltre se hai deciso di renderlo pubblico, apparir automaticamente sul portale SaaS.
